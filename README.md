Aplikasi Pengelolaan Data Buku Perpustakaan Berbasis Web

## Deskripsi Singkat
Project ini merupakan aplikasi web sederhana yang digunakan untuk mengelola data buku pada perpustakaan. Sistem ini membantu admin dalam menyimpan, mengubah, menghapus, dan mencari informasi buku secara mudah dan terstruktur.

## Struktur Database

Database memiliki 4 tabel utama:

1. Admin
- Menyimpan data akun admin untuk login.

2. Kategori
- Menyimpan kategori buku.

3. Buku
- Menyimpan data buku seperti judul, penulis, penerbit, tahun terbit, dan jumlah buku.

4. Peminjaman
- Menyimpan data transaksi peminjaman buku.

## Relasi Database

- Admin berhubungan dengan tabel Buku.
- Kategori berhubungan dengan tabel Buku.
- Buku berhubungan dengan tabel Peminjaman.
